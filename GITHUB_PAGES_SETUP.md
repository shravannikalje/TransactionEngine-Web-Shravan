## **Enable GitHub Pages (2 Easy Steps):**

### Step 1: Go to GitHub Repository Settings

1. Open: https://github.com/shravannikalje/TransactionEngine-Web-Shravan
2. Click **Settings** (top menu)
3. Scroll left to **Pages** option

### Step 2: Configure GitHub Pages

1. **Source:** Select `main` branch
2. **Folder:** Select `/docs` folder
3. Click **Save**

### Step 3: Wait & Access

- GitHub will build your site
- Wait ~1-2 minutes
- Access at: `https://shravannikalje.github.io/TransactionEngine-Web-Shravan`
